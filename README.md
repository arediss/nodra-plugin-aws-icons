# nodra-plugin-aws-icons

Official AWS architecture icons for Nodra — 305 high-fidelity AWS service icons, bundled as self-contained SVG data-URLs so they render offline with no external requests.

A plugin for [Nodra](https://github.com/arediss/Nodra).

## Install

- In the app: **Settings → Plugins** and add this plugin, or
- Install it from the plugin registry.

## Build

```sh
npm i && npm run build
```

This produces a self-contained `dist/` and a `plugin.zip` (with its sha256) you can load into Nodra.
